Distance, Displacement and Velocity

Definition of Terms


>DISTANCE
Distance can be defined as the movement of an object from one point to another.e.g the movement of a body or an object from point a to point b.etc

>DISPLACEMENT
Displacement can be defined as the distance that an object would cover under a specified direction. Displacement is a vector quantity.

>VELOCITY
Velocity can be defined as the rate of change of distance with time. Velocity is a vector quantity.
